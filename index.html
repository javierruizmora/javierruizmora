Roguelike

Juego de exploración de mazmorras con el añadido de la necesidad de recoger todos los elementos para poder pasar de nivel (funcionalidad proveniente del Pac-Man). Este juego presenta las siguientes características:

- Un único jugador.
- Escenario, obstáculos y enemigos generados aleatoriamente.
- Jugabilidad infinita.
- Dificultad exponencial.
- Muerte permanente.

  • Asignatura: Multimedia
  • Entorno de desarrollo: Unity
  • Lenguajes de programación: C#

Pac-Man (Tamagotchi)

El juego se basa en la interacción con un avatar virtual que tiene diversas necesidades interactivas que están influenciadas por el tiempo y por la interacción oportuna del usuario.
Las necesidades del avatar (energía, descanso y diversión) varían coherentemente en el tiempo (sin intervención del usuario) o en consecuencia a las interacciones del usuario. Dichas necesidades afectan al estado del avatar.
El avatar tiene un aspecto dinámico respecto a las diferentes situaciones del juego y a las necesidades y estado del avatar.
El estado y las necesidades del avatar se actualizan cada vez que se arranca el juego y durante la ejecución del mismo, utilizando para ello mecanismos de persistencia, así como varios elementos de gamificación.

  • Asignatura: Interacción Persona-Ordenador II
  • Entorno de desarrollo: Visual Studio
  • Lenguajes de programación: C++

Web VivELA

Trabajo práctico cuyo objetivo consistía en el diseño y desarrollo de una página web funcional para la solución de un problema para un cliente real. En concreto, dicho problema trataba de ayudar a las personas afectadas por la enfermedad ELA (Esclerosis Lateral Amiotrófica). En la página podremos encontrar información referente a la enfermedad y tendremos la posibilidad de colaborar con la asociación como socios, voluntarios, empresa solidaria, la realización de donativos e incluso mediante la compra de productos que la asociación oferta. También cabe la posibilidad de contacto a través de un formulario con la asociación.

  • Asignatura: Comercio Electrónico
  • Entorno de desarrollo: Prestashop
  • Lenguajes de programación: PHP, Javascript, HTML, CSS
